# CGL Pi

Implement different methods of estimating Pi.
Currently implemented:

- Monte Carlo Pi Estimate
- Bailey-Borwein-Plouffe formula
- Leibniz formula

## Usage

```bash
python main.py --method  MonteCarlo -n 100
```
